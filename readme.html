Grooves Prototype Suite

Description

A collection of browser-based sound, synth, and UI experiments focused on generative audio, ambient tools, MIDI control, and plant-inspired music synthesis. Each prototype is a standalone exploration of new interfaces, effects, or workflow concepts.

Functions

• Launch and test individual prototypes
• Explore ambient tone and pattern generators
• Control audio with MIDI and automation
• Visualize sound with real-time waveform displays
• Compare UI, presets, and effects across versions

Concept

All tools are designed for rapid creative testing, music-making, and live demo use.
The emphasis is on clarity, playability, and transparent sound design.
Code is open for remixing and extension.

Plantasia App Effects (Audio & Visual)

AUDIO EFFECTS

  • Delay – Adjustable via slider; controls delay/echo time.
  • Echo – Separate from delay; slider to control intensity/length.
  • Reverb – Stereo reverb effect, controllable.
  • Filter – Lowpass filter with LFO modulation (depth and rate adjustable).
  • Volume – Master volume control.
  • BPM / Tempo – Slider-based BPM control for melodic evolution and delay sync.
  • Base Frequency – Slider to control the central frequency for all tones.
  • Melodic Evolution – Notes/tones evolve over time, based on the Plant Pentatonic Scale or Camelot Wheel.
  • Preset System – Selecting a preset loads distinct effect values, frequencies, and color mappings.
  • Detuned Oscillators – 3+ detuned square wave oscillators per tone for analog/chorus effect.
  • Stereo Panning – Tones pan left/right in stereo field.
  • Analog Envelope – Soft attack and release for evolving tones.
  • Tape-style Pitch Wobble (tested, reverted by request) – Optional analog-style pitch drift.
  • Tape Delay (planned, not always active)—Slider control.
  • Oscillations – Number of oscillator voices, replacing “feedback.”
  • Plant Harmonic / Pentatonic Scale – All audio tuned to scientifically-backed plant frequencies.

VISUAL EFFECTS

  • Waveform Visualization
    • Animated waveform representing current audio output.
    • Sliders control vertical height and density.
    • Gradient color was mapped to the Camelot Wheel/key/frequency (updated live per tone).
    • Glow effect around the waveform.
    • Echo/ghosting trail (“visual echo”).
    • Beat-pulsed background glow.
    • Frequency markers/labels along the waveform.
    • Sinewave grid overlay (optional layer).
  • Particles (Butterflies)
    • Previously color-mapped, currently white only (by request).
    • Emerge at waveform peaks only.
    • The slider controls the quantity.
    • Animated with random size (10–200%), float/flutter behavior like butterflies.
    • Reduced density and more ambient, slower float (recent request).
    • Current: All color stripped, pure white, fewer particles.
  • UI Visual Consistency
    • All sliders and controls are visually consistent.
    • Black and white UI, with particles only in white.
    • Fullscreen toggle (added, then removed on request).
    • All slider values and states are shown in toggleable on-screen text.
    • Preset dropdown with real-time color feedback.
  • Camelot Wheel Key Feedback
    • Key/color display updates per tone live, matching the Camelot system.
  • Gradient-Enhanced Waveform
    • Each preset uses distinct waveform gradients.
    • Gradients are more intense and evolving (recent change).
